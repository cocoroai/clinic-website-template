# さくらクリニック Webサイトテンプレート

歯科・皮膚科医院向けの完全レスポンシブWebサイトテンプレート

## 技術スタック

- HTML5 / CSS3 / JavaScript
- Google Fonts (Zen Maru Gothic)

## ページ構成

- index.html - ホーム
- services.html - 診療内容
- profile.html - プロフィール
- menu.html - メニュー・料金
- blog.html - ブログ
- contact.html - お問い合わせ
- reservation.html - Web予約

## 特徴

- さくら色（ピンク）のメインカラー
- カレンダー型Web予約システム
- XSS対策・入力値検証実装

## デプロイ

- テンプレート配布用（各クライアントが独自にホスティング）
- ローカルでindex.htmlを開いて確認可能
